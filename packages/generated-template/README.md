# Generated Automation Project

Standalone Playwright pytest project generated by TC Automation Studio.

## Run

```bash
python -m pip install -r requirements.txt
python -m playwright install chromium

python -m runner.cli list-cases
python -m runner.cli run --env stg --browser chromium --all
python -m runner.cli run --env stg --browser chromium --case-key sample_case_001
python -m runner.cli run --env stg --browser chromium --all --run-id 20260530_001
python -m runner.cli rerun-failed --from-run-id 20260530_001
python -m runner.cli rerun-failed --from-run-id 20260530_001 --run-id 20260530_001_rerun
python -m runner.cli export --run-id 20260530_001 --target testrail-clone
```

## CI

This project runs independently without the GUI.

Recommended CI order:

```bash
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
python -m playwright install chromium
test -f runner/cli.py
test -f mappings/cases.yaml
python -m runner.cli list-cases
python -m runner.cli run --env stg --browser chromium --all --run-id "$CI_RUN_ID"
```

Artifacts are written to `artifacts/runs/{runId}/results.json`. Archive
`artifacts/runs` after the run.

`runner.cli run` passes the selected environment to pytest as `TC_ENV`.
